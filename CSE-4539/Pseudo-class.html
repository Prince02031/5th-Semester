CSS Pseudo-class Examples

A **pseudo-class** is a keyword added to a selector that specifies a special state of an element. Examples include when a link is hovered, clicked, visited, or when an input is focused.

Remember: a:hover must come after a:link and a:visited, and a:active must come after a:hover. Pseudo-class names are not case-sensitive.

1. :link — Unvisited Link

Unvisited links appear blue:

2. :visited — Visited Link

After clicking this once, the link will appear purple:
